# Silent9669 Tap

## How do I install these formulae?

`brew install silent9669/tap/<formula>`

Or `brew tap silent9669/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "silent9669/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
